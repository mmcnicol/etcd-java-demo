# etcd-java-demo


## start wildfly & ETCD

docker compose up --build


## to send a message, open this url in a web browser

http://localhost:8080/etcd-java-demo-1.0-SNAPSHOT/api/v1/put  
http://localhost:8080/etcd-java-demo-1.0-SNAPSHOT/api/v1/get  
http://localhost:8080/etcd-java-demo-1.0-SNAPSHOT/api/v1/delete
